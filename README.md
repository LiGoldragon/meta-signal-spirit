# meta-signal-spirit

MetaSignal contract for privileged `spirit` lifecycle and policy
surface.

This crate owns supervisor-issued orders for starting, draining, identity
registration, and bootstrap-policy reload. Ordinary psyche and intent
vocabulary remains in `signal-spirit`.

The default build is binary-only over `signal-frame`. Enable `nota-text` only
for examples, CLI-edge projection, and text round-trip tests.
